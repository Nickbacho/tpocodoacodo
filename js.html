Variables

Hay 4 formas de declarar las variables en JavaScript

  • Con la palabra reservada var: var name = 'John';
  • Con la palabra reservada let: let name = 'john';
  • Con la palabra reservada const: const PI = 3,14;
  • Sin ninguna palabra reservada: name = 'John';

Algunas aclaraciones respecto a las variables. Es una mala práctica declarar las variables sin usar una palabra reservada. Desde ECMAScript 6 se recomienda utilizar let en ves de var. La palabra const en realidad no es una variable sino una constante. En resumen, cuando declares variables utiliza let, y utiliza const para valores que no van a cambiar.

Operadores

Para abordar esta sección es importante destacar que tenemos diferentes clases de operadores.

Operadores aritméticos


Los utilizamos para realizar operaciones matemáticas básicas. a + b Suma el valor de a al valor de b. a - b Resta el valor de b al valor de a. a * b Multiplica el valor de a por el valor de b. a / b Divide el valor de a entre el valor de b. Este operador se llama Módulo a % b y devuelve el resto de la división de a entre b. Este operador es la Exponenciación A ** B y eleva A a la potencia de B.


Operadores de asignación

Estos operadores nos permiten asignar información a diferentes constantes o variables a través del símbolo =. Asignación c = a + b Asigna el valor de la parte derecha (en este ejemplo, una suma) a c. Suma y asignación a += b Es equivalente a a = a + b. Resta y asignación a -= b Es equivalente a a = a - b. Multiplicación y asignación a *= b Es equivalente a a = a * b. División y asignación a /= b Es equivalente a a = a / b. Módulo y asignación a %= b Es equivalente a a = a % b. Exponenciación y asignación a **= b Es equivalente a a = a ** b.


Operadores de comparación

Los operadores de comparación son aquellos que utilizamos para realizar comprobaciones. Estas expresiones de comparación devuelven un booleano con un valor de true o false Operador de igualdad a == b. Comprueba si el valor de a es igual al de b. Operador de desigualdad a != b Comprueba si el valor de a no es igual al de b. Operador mayor a > b Comprueba si el valor de a es mayor que el de b. Operador menor que a < b Comprueba si el valor de a es menor que el de b. Operador de igualdad estricta a===b Comprueba si el valor y ademas el tipo de dato de a es igual al de b. Operador no idéntico a !==b Comprueba si el valor y el tipo de dato de a no es igual al de b.


Operadores Lógicos

  • && (and)
  • | | (or)
  • ! (not)

Ejemplos let a = 2 let b = 2 let c = 2 if ( a == b && b == c) {console.log('hello world')} En este ejemplo se ejecuta el console.log porque se cumple la condición del if ya que las conciones son ciertas (true).
let a = 1 let b = 2 let c = 3 if ( a == b | | b == c) {console.log('hello world')} Aquí no se ejecuta el console.log ya que aunque sea una de las condiciones debería ser true para ejecutar el codigo.
let a = 1 let b = 2 if ( a != b) { console.log('hello world')} Aquí la condición se cumple ya que a es diferente que b.
